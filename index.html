Timecode Tool

TimecodeTool is a CLI that is meant to validate and calculate timecode spans. It is an open-source project written in Go by Marc Leoanrd. is released for free under the MIT License.

  • Download
  • View on GitHub

Validate

Enter a timecode to see if it is valid. Greate for scripting and checking user input.

TimecodeTool --fps=29.97 "00:07:00;00"

Timecode Spans

Enter a span to see if it is valid. Greate for scripting and checking user input.

TimecodeTool --fps=23.98 "00:01:00:00" "00:04:00:00"

Calculate

Enter a series of calculations to return a final run length.

TimecodeTool --fps=23.98 "00:01:00:00" + "00:04:00:00" + 2 - "00:00:00:10"
